aquest projecte es una demostracio de com accedir a les dades de una api rest amb token jwt amb java i una interficia grafica swing. la api rest esta en la seguent ruta (es te uqe poder modificar en cofiguracio per adaptar a futurs canvis de ruta) http://localhost:8080/ 
per exemple la ruta de registre seria http://localhost:8080/api/auth/register o la de recuperar la informacio del usuari actual http://localhost:8080/api/users/me
aqui s'implementara a mode de exemple les opcions de registre d'usuari, login, edicio, llistat dels usuaris amb els dos rols posibles (administradors i usuaris),
els comentaris i els noms de les variables s'escriuran en catala.
